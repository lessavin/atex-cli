Domain edit record:
To make edit request, you need to add elid={record_id} parameter to the add record request.

Example A:
https://my.atex.ru/billmgr?elid={record_id}&plid={domain_id}&func=domain.dnsrecords.edit&out=xml&authinfo={login}:{password}&dns_type=A&dns_name={dns_name}&ipv4={ip}&clicked_button=ok&sok=ok&ttl=3600

DNSSEC:

https://my.atex.ru/billmgr?plid={domain_id}&func=domain.dnsrecords.dnssec&out=xml&authinfo={login}:{password}
