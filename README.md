# atex-cli

atex-cli - [Atex.ru](https://atex.ru/) command line tools.

## Synopsis

atex-cli list  
atex-cli add  
atex-cli get  
atex-cli edit  
atex-cli auto  
atex-cli dnssec  
atex-cli dnssec-enable  
atex-cli soa

## Description

atex-cli is a collection of tools for accessing Atex service from a command line.

The project's primary goal is to create an application with a command-line interface (CLI) identical to the graphical
user interface (GUI) provided by the atex.ru personal account web application.

A secondary goal of the project is to provide an application programming interface (API) for creating similar
application.

## Tools Overview

| Tool                   | Description          |
|------------------------|----------------------|
| atex-cli list          | domain(s) list       |
| atex-cli add           | domain add record    |
| atex-cli get           | domain get records   |
| atex-cli edit          | domain edit record   |
| atex-cli delete        | domain delete record |
| atex-cli auto          | DNS auto settings    |
| atex-cli dnssec        | get DNSSEC key       |
| atex-cli dnssec-enable | DNSSEC enable        |
| atex-cli soa           | edit SOA             |
